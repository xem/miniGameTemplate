# MiniGameTemplate

Modern, editable HTML5 game template in 100 lines of code:

- Standalone
- SVG favicon
- Twemoji polyfill
- Disable text select & touch gestures crap
- Keyboard input manager
- State machine + 60fps game loop/click handler 
- 0.8kb minified
